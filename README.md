# Коллекционные приложения для Android

Приложения для коллекционеров монет, банкнот и других предметов коллекционирования.

<p align="center">
  <img src="https://green-enoz.github.io/images/about_phone.png" width="400" alt="Скриншот приложений">
</p>

---

## 📱 Приложения

### 🪙 Мои Монеты

Каталоги монет России, СССР и других стран, учет коллекции, описание монет, тиражи и характеристики.

### 💵 Мои Банкноты

Удобный каталог банкнот с возможностью вести собственную коллекцию и хранить информацию о купюрах.

### 📚 Моя Коллекция

Универсальное приложение для создания собственных каталогов и учета любых коллекционных предметов.

---

## ✨ Возможности

* Каталоги коллекций
* Создание собственных каталогов
* Добавление и редактирование элементов
* Учет предметов в коллекции
* Хранение изображений и описаний
* Поиск и удобная навигация
* Работа полностью на устройстве

---

## ℹ️ Информация о коллекционных предметах

Для каждого элемента коллекции можно хранить:

* изображения
* описание
* характеристики
* дату выпуска
* тираж
* заметки и дополнительную информацию
